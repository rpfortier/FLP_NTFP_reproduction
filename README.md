# Climate affects the reproductive phenology of two non-timber forest product species in the southwestern Amazon 

Contact: Riley P. Fortier (fortier.riley@gmail.com)

This repository contains code and associated data to analyze phenology and climate effects on the reproduction of two important NTFP species in the Amazon. A manuscript is currently in review.

All R scripts run successfully using R software version 4.5.0 on Windows 11 and macOS Tahoe and should work on all standard operating systems. The R script "NTFP_phenology_analysis_BE.R" is required to run first in order to load necessary dataframes and objects used by "NTFP_phenology_analysis_MF.R".

## *Data*
* BE_phenology_data.csv
  + Phenology data for *Bertholletia excelsa*.
* MF_phenology_data.csv
  + Phenology data for *Mauritia flexuosa*.
* BE_locations.csv
  + Locality data for *Bertholletia excelsa* trees, including forest type.
* ASA_FLP_weather_climate_data_MASTER.xlsx
  + Raw weather station data from Finca las Piedras.

## *R scripts*
* NTFP_phenology_analysis_BE.R
  + Data analysis for *Mauritia flexuosa*
* NTFP_phenology_analysis_MF.R
  + Data analysis for *Bertholletia excelsa*
